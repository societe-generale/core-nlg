    Freeze synonyms #

    You can freeze the synonyms with the corresponding argument of the NlgTools class (or the Document class if you use CoreNLG as a framework).

    In freeze mode, when using synonyms, the first possibility will always be drawn. This allows you to test the non-regression of your app on a specific use case. However you should keep in mind it will not help you to detect changes on other branches than the first ones.
